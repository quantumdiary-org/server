# QuantumDiary Server - CHANGELOG

## v1.0.0 - 2026-01-17

### Особенности

- **Полная реализация NSWebAPI** - все методы из @nsapi-web/** полностью реализованы
- **Поддержка нескольких типов API** - NSWebAPI, NSMobileAPI, DevMockAPI
- **Динамический выбор типа API** - тип API передается при аутентификации и сохраняется в сессии
- **Браузерная аутентификация** - поддержка входа через Playwright (опционально)
- **Унифицированный трансформер** - все данные приводятся к единому формату QD-1
- **Поддержка различных баз данных** - PostgreSQL, MariaDB, MySQL, SQLite
- **Многоуровневая аутентификация** - разделение токенов для клиента и прокси-сервера
- **Интеграция с кэшем** - поддержка Redis и in-memory кэширования
- **Отказоустойчивость** - резервное кэширование при недоступности внешних систем
- **Rate limiting** - защита от чрезмерного использования API
- **Мониторинг состояния** - health check эндпоинты для проверки работоспособности
- **Автоматическое обновление кэша** - сервис для автоматического обновления данных
- **Резервные копии данных** - при недоступности API используются закэшированные данные
